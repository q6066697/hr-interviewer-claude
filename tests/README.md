# Результаты тестирования ассистента

Здесь собраны выводы по трём тестовым прогонам HR-ассистента **TechHire Interviewer**. Тесты проводились в Claude Projects для проверки корректности промпта и базы знаний.

## Сценарии

| Файл | Сценарий | Ожидание | Факт | Результат |
|---|---|---|---|---|
| `test_1_strong_candidate.txt` | Сильный кандидат: глубокие технические ответы, зрелые soft skills | Рекомендовать, тех. балл ≥ 4.0 | Тех. 4.75 / Soft 5.0 | ✅ |
| `test_2_weak_candidate.txt` | Слабый кандидат: поверхностные ответы, красные флаги | Не рекомендовать, тех. балл < 3.0 | Тех. 2.0 / Soft 1.7 | ✅ |
| `test_3_border_candidate.txt` | Пограничный: сильный Python, слабая БД, нормальные soft skills | Рекомендовать с оговорками, тех. балл 3.0–3.9 | Тех. 3.25 / Soft 4.0 | ✅ |

## Что проверялось

- Соблюдение структуры из 5 этапов интервью
- Использование базы знаний при оценке (ссылки на красные флаги, ценности компании, требования вакансии)
- Корректная классификация кандидатов по правилу финальной рекомендации
- Соблюдение правил (не подсказывать ответы, не обсуждать зарплату, нейтральный тон)
- Адекватная работа в пограничной зоне с нюансированной оценкой
